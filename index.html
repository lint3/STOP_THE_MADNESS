Watch out!

⚠️ Think about controlled data. All processing in this tool occurs client-side - no data is transmitted outside your local browser window. That means you can use it with proprietary data. But should you take my word for it? Consider the following mitigations you might take to reduce the possibility of leakage:

  • Sanitize your BOM file before using it with this tool. (Remove CPNs, MPNs, or other data before importing)
  • Just don't import a BOM file. This tool is still useful for deduplicating, formatting, and comparing lists of IPNs, refdes, etc. These operations don't require a BOM import.
  • If you're really reading this warning, click the yellow triangle emoji to dismiss this dialog when you're done.
  • Using your browser's devtools, review the code to make sure no network requests will be made.

By the way, these concerns should apply to all software you use on your work computer, not just browser-based tools like this one. Macro-enabled Excel files, python scripts, screenshot utilities, and other software pose these very same risks!

Help

Features

Ranges
Input ranges like R1-R8 are expanded automatically. Enable the Range toggle to collapse output back to range notation.
Partial matches
An item in the output area may not be fully represented in the input area (e.g. a refdes is in input but has siblings of the same IPN that were not mentioned). Enable Partial to flag these.
Output delimiter
Customize the separator between output items (use \n for line break, \t for tab).
Diff highlighting
Colors tints items by how many panels contain them (red = unique, yellow = partial match). DiffOnly hides items present in all panels.
BOM import
Load an .xlsx file and map its columns; converting between data types is enabled once a BOM is loaded.

Use Cases

Deduplicate / normalize a list
Paste a messy list → get sorted, deduped output.
Convert an IPN list to find numbers
Import BOM → paste IPN list → set IPN input → FN output.
Compare refdes list to find number list
E.g. X-Ray requirements (assembly drawing) vs. X-Ray program: Import BOM → paste refdes in one panel + FN list in another → set input/output types → compare.
Compare machine programs to work instructions
Paste each list in its own panel → set types → diff highlights mismatches.
